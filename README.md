# NPS Research NLW

Next Level Week #04 - Node trail project

#### To run this project:

- clone this repo: `git clone https://github.com/paolamonteiro/nps-research-nlw.git`
- move to api/
- run `npm install` or `yarn install`
- run `npm run dev` or `yarn dev`

#### Main Tools:

- Typescript
- Node.js
- Express
- Sqlite3
- TypeORM
